Usage with Promises This package also optionally works with promises. Just omit the callback parameter and the search method returns a promise object containing the response: var Spotify = require('node-spotify-api'); var spotify = new Spotify({ id: , secret: }); spotify .search({ type: 'track', query: 'All the Small Things' }) .then(function(response) { console.log(response); }) .catch(function(err) { console.log(err); }); Request request can be used to make API requests to any Spotify endpoint you supply.
